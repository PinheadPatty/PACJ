PACJ is an open-source software framework built on ROS 2 for managing autonomous robotic tasks — including drone flight, ground rover navigation, 3-D mapping, localization, and module manipulation.

View on GitHub Learn More

What PACJ Does

A unified framework that bridges air and ground autonomy.

🚁

Drone Control

Full PX4-based flight control with offboard setpoint planning, battery monitoring, and autonomous mission execution.

🤖

Rover Navigation

Ground rover driver with servo control, servo-based locomotion, and communication health checks over the robot network.

🗺️

3-D Mapping

SLAM-driven occupancy mapping using depth data from the Orbbec G330 RGB-D camera and ROS 2 nav2 integration.

📍

Localization

Visual-inertial odometry relay and TF broadcasting keep all agents localised within a shared map frame in real time.

🔲

Module Manipulation

High-level planner coordinates grabbing and placement of physical modules, enabling collaborative construction tasks.

📷

RGB-D Vision

Orbbec ROS 2 SDK integration provides live point clouds and compressed image streams to every node in the graph.

System Architecture

Three cooperating agents share a single ROS 2 topic graph.

Agent

Drone

PX4-powered aerial platform. Runs drone_driver and drone_planner nodes.

Agent

Rover

Ground vehicle running rover_driver with servo-based actuation and a ping health monitor.

Operator

Laptop

Runs RViz 2, visualises the live map, and provides the operator interface via interactive_setpoint.

Perception

Orbbec G330

Depth camera providing point clouds, IMU data, and compressed RGB streams to the ROS 2 graph.
